# Курсовой проект: сбор данных о компаниях и вакансиях с сайта hh.ru и хранение в PostgreSQL

## Цель проекта

    Цель данного проекта — получение данных о работодателях и их вакансиях с сайта hh.ru, 
    организация их хранения в базе данных PostgreSQL и разработка удобного инструмента для анализа собранных данных.

## Перед запуском проекта:

    -скопируйте .env.example в .env

    -заполните .env реальными значениями

## Установка:
  1. Клонируйте репозиторий:

    https://github.com/AleksChek26/CourseWork_3

  2. Установите зависимости:

    poetry install

## Использование:

    модуль main.py реализует логику проекта

## Документация:
